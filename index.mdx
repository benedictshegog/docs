## Overview

The Granular Energy Data Upload API allows you to upload and manage energy generation and consumption data for your organization. This API supports:

- **Generation Data Upload** - Upload metering data from renewable energy generation sources
- **Consumption Data Upload** - Upload energy consumption data for tracking and matching
- **ESNA Integration** - Manage ESNA (Exempt Supplier Notification Agent) related data and contracts

## Getting Started

To use the Granular Energy API, you'll need:

1. An API key with the appropriate scopes for your use case
2. Valid metering point references configured in your organization
3. Energy data in the required CSV format

Contact [support@granular-energy.com](mailto:support@granular-energy.com) to request API access.

## Rate Limits

Most upload endpoints are subject to rate limits to protect service availability:

- **File Upload Endpoints**: Maximum 5 calls per minute
- **File Size Limit**: 200MB maximum
- **Measurements per File**: 1 million maximum
